=====================

Pathfinder is a tool for the visual exploration of paths in large graphs.

Installation
------------

```
git clone https://github.com/phovea/pathfinder.git
cd pathfinder
npm install
```

Testing
-------

```
npm test
```

Building
--------

```
npm run build
```
