Pathfinder

Pathfinder is a visual analysis tool for the exploration of paths in large networks that was built with Caleydo Web.
For further information, please visit our publication page.


For best compatibility, please make sure to use Pathfinder with Google Chrome

Examples

Co-Author Network

We built a co-author network from ACM CHI and IEEE TVCG publications from the DBLP computer science bibliography. This network has 34,384 nodes and 45,347 edges.

Try it!

Pathway Network

We constructed a pathway network from KEGG pathways. It describes interactions of genes and metabolites. We also load additional mRNA expression and copy-number variation data from Cancer Cell Line Encyclopedia. This network has 11,355 nodes and 71,093 edges.

Try it!
